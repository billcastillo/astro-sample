## Para san ito?

Para mabawasan ang lag kapag maraming players / characters sa mga events tulad ng cw, suryun, pk, etc..

## Instructions:

- Go to Ran Folder > data > skinobject
- Search cos_min
- Cut and paste to backup or any folder
- Press F9 to toggle (on/off)
- Move to another map to refresh
